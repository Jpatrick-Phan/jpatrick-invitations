# Design System & UI Guidelines

## 1. Design Philosophy
**"Elegant, Soft, Flowing"** for Wedding.
**"Formal, Bright, Energetic"** for Graduation.

## 2. Color Palette (Wedding Default)
-  **Primary Gold**: `#D4AF37` (Classic Gold)
-  **Soft Cream**: `#FFFDF5` (Background)
-  **Text Dark**: `#2C3E50` (Readable text)
-  **Text Light**: `#8B7E74` (Subtitles)
-  **Accent Pink**: `#F8E1E7` (Highlights)

## 3. Typography
-  **Script/Display**: `Great Vibes` or `Pinyon Script` (For Names, Headers).
-  **Serif**: `Playfair Display` (For Section Titles).
-  **Sans-Serif**: `Inter` or `Montserrat` (For Body text, Dates, Info).

## 4. Spacing & Layout
-  **Mobile First**: Design for 375px width (iPhone SE) first.
-  **Breathing Room**: Generous padding (e.g., `padding: 4rem 1rem`) between sections.
-  **Container**: Max-width 600px for text content to ensure readability.

## 5. Animations (The "WOW" Factor)
-  **Scroll Reveal**: Elements should fade up (`translateY(20px) -> 0`) as they enter viewport.
-  **Parallax**: Background images move slightly slower than scroll.
-  **Floating**: Subtle bounce for "Down Arrow" or floating elements.
-  **Transitions**: All hover states must have `transition: all 0.3s ease`.

## 6. Components
-  **Buttons**: Rounded pill shape, slight shadow, scale up on hover.
-  **Cards**: Frosted glass effect (`backdrop-filter: blur(10px)`) for overlays on images.
-  **Inputs**: Minimalist transparent background with bottom border.
